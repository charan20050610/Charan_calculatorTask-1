# Charan_calculatorTask-1About Calculator

This Calculator Web Application is a simple and responsive project developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations such as addition, subtraction, multiplication, division, and percentage calculations through an interactive user interface.

The project features a clean design with a display screen for showing user input and calculated results. CSS Grid is used to organize the calculator buttons, while JavaScript handles user interactions and performs calculations dynamically. Error handling is included to manage invalid inputs and improve usability.

Features:
Basic arithmetic operations (+, −, ×, ÷)
Percentage (%) calculation
Responsive and user-friendly interface
Real-time display updates
Error handling for invalid expressions
Built with HTML, CSS, and JavaScript
